# agent.ts 注释规范化重构设计

## 1. 目标

对 `src/agent/agent.ts` 进行彻底的注释规范化，统一注释风格，补全缺失的中文注释。

## 2. 变更范围

### 2.1 注释风格统一

- 统一使用 `/** 中文 */` 格式
- 类型/接口/类注释在声明上方
- 方法注释在方法上方
- 字段注释在字段上方

### 2.2 待添加/修改注释的内容

| 位置 | 注释类型 | 目标 |
|------|---------|------|
| `QueueMode` 类型 | 中文注释 | 说明队列模式枚举 |
| `MutableAgentState` 类型 | 中文注释 | 说明可变状态类型 |
| `ActiveRun` 类型 | 中文注释 | 说明活动运行类型 |
| `AgentOptions` 接口 | 改为中文 | 保持现有结构 |
| `PendingMessageQueue` 类 | 添加中文注释 | 说明队列功能 |
| `Agent` 类 | 改为中文 | 保持现有结构 |
| 所有 public 方法 | 改为中文 | `prompt`, `continue`, `subscribe` 等 |
| 所有 getter/setter | 添加中文注释 | `state`, `steeringMode`, `signal` 等 |
| 所有 private 方法 | 改为中文 | `normalizePromptInput`, `runPromptMessages` 等 |

## 3. 不变更项

- 不创建新文件
- 不拆分 `Agent` 类
- 不改变任何类型定义
- 不改变任何代码逻辑
- 不改变导出顺序

## 4. 风险评估

| 风险 | 缓解措施 |
|------|---------|
| 注释改动引入错误 | 仅文本改动，无结构变更 |
| 遗漏部分注释 | 按本设计文档逐项检查 |

## 5. 验收标准

- 所有类型、接口、类都有中文注释
- 所有方法、字段都有中文注释
- 注释风格统一为 `/** 中文 */` 格式
- TypeScript 编译无错误
